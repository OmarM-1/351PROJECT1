HOW TO RUN PROGRAM: 
- Compile sender.cpp and recv.cpp
    - Commands:
      - g++ sender.cpp -o sender -lrt
      - g++ recv.cpp -o recv -lrt
- Run the reciever:
    - ./recv
- In a seperate terminal run the sender file to send the file to the reciever:
    - ./ sender file.txt
